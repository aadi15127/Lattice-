# 🕸️ Lattice | CSS Grid Architect

Lattice is a real-time **CSS Grid visualizer** that allows developers to design complex layouts with ease. Adjust columns, rows, gutters, and alignment, then export your code instantly.



## 🚀 Features
- **Visual Grid Editor:** Click items to expand them across columns or rows.
- **Real-time Code Generation:** Instant CSS output as you tweak settings.
- **Export Options:** Download `.css` files or email code directly using EmailJS.
- **Local History:** Save your favorite layouts locally to your browser.

## 🛠️ Tech Stack
- HTML5 / CSS3 (Grid & Flexbox)
- Vanilla JavaScript
- EmailJS Integration

## 📦 Installation
Simply clone the repository and open `index.html` in any modern web browser.
